# Ingest Sources

## 1. Teams message
営業・社内確認依頼・雑な依頼の入口。

例:
- PLまだ？
- SI-224も確認して
- Customer C、AIR必要か見て

## 2. Email body
仕入先・Forwarder・営業からのメール本文。

例:
- ETA changed
- invoice will be reissued
- PL attached

## 3. Attachment / document
INV / PL / BL / SI / Booking などの書類。

例:
- Commercial Invoice
- Packing List
- Bill of Lading
- Shipping Instruction
